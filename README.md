# Life Search Web Demo

iPhone Safari için cihaz içinde çalışan Life Search PWA denemesidir.

- Fotoğraflar sunucuya yüklenmez.
- MobileNet ile nesne, baskın renk ve çekim tarihi aranır.
- Safari → Paylaş → Ana Ekrana Ekle ile uygulama gibi kullanılabilir.

Canlı demo: https://ozkandmrcn.github.io/life-search-web-demo/
